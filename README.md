# AutoFlow-Pro
Automate batch image &amp; video generation on Google Flow. Visual workflows, smart tasks, prompt templates, and auto-download built in.

Tired of submitting prompts one by one on Google Flow? AutoFlow Pro turns Google Flow (labs.google/fx) into a powerful batch generation engine — submit hundreds of prompts automatically while you focus on what matters.

WHAT IT DOES

AutoFlow Pro is a productivity extension that automates prompt submission on Google Flow. It does NOT run AI models — Google Flow handles all image and video generation. AutoFlow Pro simply automates the repetitive clicking and typing.

KEY FEATURES

Batch Prompt Generation
- Submit multiple prompts automatically, one after another
- Import prompts from .txt files or paste a list
- Per-prompt reference images with @mention syntax
- Humanized typing mode with random delays to stay safe

Visual Workflow Builder
- Drag-and-drop node editor for complex generation pipelines
- Chain generate, upscale, download, and delay nodes
- Parallel or sequential execution
- Video frame inputs from upstream image nodes

Smart Tasks
- Save prompt + settings as reusable tasks
- Run all tasks with one click (sequential or parallel)
- Per-task auto-download with custom subfolders

Prompt Templates
- Browse curated prompt templates by category
- Filter by style, difficulty, and media type
- One-click apply to generation

Auto-Download
- Automatically download generated images and videos
- Customizable filename templates: [Date]_[Project]_[Prompt]_[Index]
- Organized subfolders per task, workflow, or module
- Choose resolution: 1K, 2K for images / 720p, 1080p for video

Multi-Angle Camera (Angles)
- Generate the same subject from multiple camera angles
- Orbit controls for rotation, tilt, and zoom
- Preset-based angle modifiers

Image Effects
- Apply 28 effects across 5 categories: Color, Light, Weather, Artistic, Texture
- Adjustable intensity slider
- Batch apply effects to reference images

Album & Reference Images
- Organize reference images into albums
- @mention images by name in prompts
- Cross-project reference image support

HOW IT WORKS

1. Open Google Flow (labs.google/fx) in Chrome
2. Click the AutoFlow Pro icon to open the sidebar
3. Type or import your prompts
4. Set your preferences (model, ratio, quantity)
5. Click Generate — AutoFlow Pro handles the rest
